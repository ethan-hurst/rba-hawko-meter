<task type="auto">
  <name>Task 1: Wire pytest-cov addopts and add dev dependencies</name>
  <files>pyproject.toml, requirements-dev.txt</files>
  <action>
1. Edit `pyproject.toml` — update the `[tool.pytest.ini_options]` addopts array from:
   ```toml
   addopts = ["--strict-markers"]
   ```
   to:
   ```toml
   addopts = [
       "--strict-markers",
       "--cov=pipeline",
       "--cov-report=term-missing",
       "--cov-report=json:.coverage.json",
   ]
   ```
   This scopes coverage to the `pipeline/` production code only (test files excluded), outputs a terminal report with missing line numbers, and writes `.coverage.json` at the project root for the coverage check script (Plan 18-02). The `:DEST` suffix on `json:.coverage.json` is required — without it, pytest-cov writes `coverage.json` (no dot prefix). Do NOT add `--cov-fail-under` — per-module enforcement is handled by the script in Plan 18-02, not by a global threshold.

2. Edit `requirements-dev.txt` — add two new lines after the existing entries, preserving the existing comment and format:
   ```
   pytest-cov>=4.0
   pytest-mock>=3.15
   responses>=0.26
   ```
   Use minimum version bounds (not pinned) per locked decision. pytest-cov 7.0.0 is already installed but missing from the manifest — adding it ensures reproducible installs.

3. Install the new dependencies:
   ```bash
   pip install -r requirements-dev.txt
   ```
  </action>
  <verify>
    <automated>python -c "import pytest_mock; import responses; print('OK')" && pytest tests/python/test_smoke.py -x --tb=short 2>&1 | grep -E "(PASSED|coverage|pipeline/)"</automated>
    <manual>Confirm terminal shows per-module coverage table and .coverage.json exists at project root</manual>
  </verify>
  <done>
  - pyproject.toml addopts contains --cov=pipeline, --cov-report=term-missing, --cov-report=json:.coverage.json
  - requirements-dev.txt lists pytest-cov>=4.0, pytest-mock>=3.15, responses>=0.26
  - `pip install -r requirements-dev.txt` succeeds
  - `python -c "import pytest_mock; import responses"` succeeds
  - `pytest tests/python/test_smoke.py` produces coverage output in terminal
  </done>
</task>
