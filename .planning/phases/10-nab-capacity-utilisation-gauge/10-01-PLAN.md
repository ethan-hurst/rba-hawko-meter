  <action>
Complete rewrite of `pipeline/ingest/nab_scraper.py`. Replace the existing stub entirely.

**URL Discovery (NAB-02):**
- `discover_latest_survey_url(session)`: Crawl two tag archive URLs (`business.nab.com.au/tag/economic-commentary` and `business.nab.com.au/tag/business-survey`). Parse all `<a>` tags, find first href containing "monthly-business-survey" (case-insensitive). Return absolute URL or None. NEVER construct URLs from date templates for current-month discovery.
- Define `TAG_ARCHIVE_URLS` list and `NAB_BASE = "https://business.nab.com.au"` as module constants.

**HTML Extraction (NAB-01):**
- `extract_capacity_from_html(html_bytes)`: Parse with BeautifulSoup/lxml, search all `<p>`, `<li>`, `<div>` tags. Apply regex `r'[Cc]apacity utilisa?tion[^.]*?([\d]+\.?\d*)%'` (handles both AU/US spelling via `sa?tion`). Return float or None.
- `CAPACITY_REGEX` as module-level compiled regex with `re.IGNORECASE`.

**PDF Fallback (NAB-05):**
- `get_pdf_link(html_bytes)`: Extract first PDF link from article HTML anchors. Make absolute URL if relative.
- `extract_capacity_from_pdf(pdf_bytes)`: Use pdfplumber (already in requirements.txt), scan first 6 pages, apply same CAPACITY_REGEX. Return float or None.

**Idempotency:**
- `_current_month_already_scraped(output_path)`: Read existing CSV, check if latest date matches current year+month. Return True to skip. Mirrors corelogic_scraper.py pattern.

**Backfill:**
- `backfill_nab_history(session, months=12)`: Loop over last 12 months. For each month, try 3 URL patterns by construction (Pattern A: `/tag/economic-commentary/nab-monthly-business-survey---{month}-{year}`, Pattern B: `/{root}/nab-monthly-business-survey-{month}-{year}/`, Pattern C: same without trailing slash). Extract from HTML, fall back to PDF for each. Append successful extractions to CSV. This is the ONLY place URLs are constructed from templates — for historical backfill where tag archive does not list old articles.
- Define `MONTH_URL_PATTERNS` as list of lambda functions.

**Main scraper:**
- `scrape_nab_capacity()`: Check idempotency → discover URL → fetch article → HTML extract → PDF fallback → return single-row DataFrame with columns `[date, value, source]`. Date = first of reference month (e.g., 2026-01-01 for Jan 2026 data). Source = "NAB Monthly Business Survey". Return empty DataFrame on any failure (log warning, never raise).
- `fetch_and_save()`: Orchestrate scrape + CSV append. Never raises. Returns status dict `{status: success/failed, rows: N}`. If CSV doesn't exist and has <3 rows, trigger backfill first.

**Key constraints:**
- Use `create_session(retries=3, backoff_factor=0.5, user_agent=BROWSER_USER_AGENT)` from `pipeline.utils.http_client`.
- Use `append_to_csv()` from `pipeline.utils.csv_handler` for dedup-safe CSV writes.
- Import constants from `pipeline.config`: `DATA_DIR`, `BROWSER_USER_AGENT`, `DEFAULT_TIMEOUT`.
- Output file: `DATA_DIR / "nab_capacity.csv"`.
- Only write to CSV when value is a valid float — NEVER write NaN/None rows.
- Logging: use module-level `logger = logging.getLogger(__name__)`.
