**Fields to add to the gauge entry when `name == 'business_confidence'`:**

1. **`long_run_avg`**: Compute dynamically from all available values in z_df. Use `z_df['value'].dropna().mean()`. If fewer than 2 data points, default to 81.0 (established NAB research figure). Round to 1 decimal.

2. **`direction`**: Compare last two values in z_df. If delta > +0.5: "RISING". If delta < -0.5: "FALLING". Otherwise: "STEADY". If only 1 data point, set direction to None (omit from entry — frontend handles gracefully by showing level only).

3. **`data_source`**: Set to `"NAB Monthly Business Survey"` (hardcoded — unlike housing which has multiple sources, NAB only has one).

4. **`raw_unit`**: Override from `'% YoY'` to `'%'` — capacity utilisation is an absolute percentage, not a YoY change.

5. **Staleness check**: The 45-day threshold is a frontend concern. The engine already writes `staleness_days` to the entry. No special staleness logic needed in engine.py beyond what's already there.

**Implementation:**
```python
if name == 'business_confidence':
    import pandas as _pd
    # Load raw CSV for direction computation (z_df may be resampled)
    csv_path = DATA_DIR / config.get('csv_file', '')
    if csv_path.exists():
        raw_df = _pd.read_csv(csv_path)
        raw_df['date'] = _pd.to_datetime(raw_df['date'])
        raw_df = raw_df.sort_values('date')
        all_values = raw_df['value'].dropna()

        # Long-run average
        long_run_avg = float(all_values.mean()) if len(all_values) >= 2 else 81.0
        entry['long_run_avg'] = round(long_run_avg, 1)

        # Direction: month-over-month change
        if len(all_values) >= 2:
            curr_val = float(all_values.iloc[-1])
            prev_val = float(all_values.iloc[-2])
            delta = curr_val - prev_val
            if abs(delta) <= 0.5:
                entry['direction'] = 'STEADY'
            elif delta > 0:
                entry['direction'] = 'RISING'
            else:
                entry['direction'] = 'FALLING'
        # If only 1 data point, omit direction (handled gracefully by frontend)

    entry['data_source'] = 'NAB Monthly Business Survey'
    entry['raw_unit'] = '%'
```
