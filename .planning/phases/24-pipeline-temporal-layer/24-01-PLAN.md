<task type="auto">

  <action>
Create `pipeline/normalize/archive.py` with three functions that make all tests pass.

**save_snapshot(status_dict, snapshots_dir, max_entries=52):**
- `snapshots_dir = Path(snapshots_dir)` then `snapshots_dir.mkdir(parents=True, exist_ok=True)`
- Format today as `datetime.utcnow().strftime('%Y-%m-%d')`
- Write `status_dict` to `snapshots_dir / f"{today}.json"` with `json.dump(..., indent=2)`
- Load or create `index.json` at `snapshots_dir / "index.json"` with structure `{"snapshots": []}`
- If today not in list, append and sort
- While len > max_entries: pop index 0 (oldest), unlink the corresponding file if it exists
- Write updated index.json

**read_previous_snapshot(snapshots_dir, min_age_days=5):**
- `snapshots_dir = Path(snapshots_dir)`
- If `index.json` doesn't exist, return None
- Load index, iterate `reversed(index["snapshots"])`
- Parse each date string with `datetime.strptime(date_str, '%Y-%m-%d').date()`
- Compute age as `(datetime.utcnow().date() - snap_date).days`
- If `age >= min_age_days` AND the snapshot file exists, load and return it
- If no candidate found, return None

**inject_deltas(status_dict, previous_snapshot):**
- If `previous_snapshot is None`: return immediately (no-op)
- Overall: get `previous_snapshot['overall']['hawk_score']`, if not None: set `status_dict['overall']['previous_hawk_score']` and `status_dict['overall']['hawk_score_delta'] = round(curr - prev, 1)`
- Per-gauge: iterate `status_dict['gauges'].items()`, get matching gauge from `previous_snapshot['gauges']` via `.get(name)`. If prev gauge exists and has 'value': set `previous_value`, compute `delta = round(curr - prev, 1)`, set `direction` ('up'/'down'/'unchanged')

Module docstring: "Snapshot archival and temporal delta injection for the data pipeline."

Imports needed: `json`, `pathlib.Path`, `datetime.datetime` only (all stdlib).

Run tests to confirm GREEN state: `pytest tests/python/test_archive.py -v` should show all passing.
