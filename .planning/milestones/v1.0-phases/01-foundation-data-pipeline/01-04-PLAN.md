Key facts from research:
- ABS Building Approvals dataflow ID is `BA_GCCSA` (NOT "BA" or "BUILDING_APPROVALS")
- API URL: `https://data.api.abs.gov.au/data/ABS,BA_GCCSA/all`
- Data format: same SDMX CSV as other ABS sources (TIME_PERIOD, OBS_VALUE columns)
- Monthly release schedule (latest data always ~2 months lag)
- The existing `fetch_abs_series()` function in abs_data.py handles all ABS Data API requests generically, so only config + wrapper function needed
