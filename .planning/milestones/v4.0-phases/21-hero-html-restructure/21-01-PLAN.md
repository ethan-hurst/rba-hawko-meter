<task type="auto">
  <name>Task 2: Update gauge-init.js to render hero card content and wrap Plotly in rAF</name>
  <files>public/js/gauge-init.js</files>
  <action>
Modify the `initGauges()` function's `.then()` callback in gauge-init.js. The current callback renders the hero gauge, verdict, jump link, ASX table, freshness, and metric gauges. Update it to:

**1. Wrap hero gauge creation in requestAnimationFrame:**
Replace the direct call:
```javascript
GaugesModule.createHeroGauge('hero-gauge-plot', data.overall.hawk_score);
```
With:
```javascript
requestAnimationFrame(function () {
  GaugesModule.createHeroGauge('hero-gauge-plot', data.overall.hawk_score);
  requestAnimationFrame(function () {
    var heroEl = document.getElementById('hero-gauge-plot');
    if (heroEl && heroEl.data) {
      Plotly.Plots.resize('hero-gauge-plot');
    }
  });
});
```

**2. Add hawk score display rendering (after verdict render, before jump link):**
```javascript
// Render hawk score number in hero card
var scoreDisplay = document.getElementById('hawk-score-display');
if (scoreDisplay) {
  scoreDisplay.textContent = Math.round(data.overall.hawk_score) + '/100';
}
```

**3. Add zone border colour (after score display):**
```javascript
// Set zone-coloured top border on hero card
var heroCard = document.getElementById('hero-card');
if (heroCard) {
  heroCard.style.borderTopColor = GaugesModule.getZoneColor(data.overall.hawk_score);
}
```

**4. Change freshness target from 'data-freshness' to 'hero-freshness':**
Replace:
```javascript
InterpretationsModule.renderStalenessWarning(
  'data-freshness', data.generated_at
);
```
With:
```javascript
InterpretationsModule.renderStalenessWarning(
  'hero-freshness', data.generated_at
);
```

**5. Add hero card animation trigger (at the end of the success callback, after metric gauges):**
```javascript
// Animate hero card entry (only on success, respect reduced motion)
if (heroCard && !window.matchMedia('(prefers-reduced-motion: reduce)').matches) {
  heroCard.classList.add('hero-animate-in');
}
```

The final order of operations in the .then() callback should be:
1. `requestAnimationFrame` wrapping `createHeroGauge` + nested rAF for `Plotly.Plots.resize`
2. `renderVerdict('verdict-container', data.overall)`
3. Set `#hawk-score-display` textContent
4. Set `#hero-card` borderTopColor
5. Render calculator jump link
6. `renderASXTable`
7. `renderStalenessWarning('hero-freshness', data.generated_at)`
8. `renderMetricGauges(data.gauges)`
9. Render missing indicator cards
10. `renderCalculatorBridge`
11. Add `hero-animate-in` class if reduced motion not preferred

Do NOT use innerHTML. Use createElement/textContent/appendChild only.
